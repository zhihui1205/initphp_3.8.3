工具描述：

主要是用来在开发的时候调试BUG记录日志用。属于InitPHP框架工具类，需要通过$this->getUtil()方法获取

使用方法：

$log = $this->getUtil('log');

接口描述：

接口：$error->write($message, $log_type = 'DEBUG') 写日志
参数 类型 是否必须 描述
$message String 是 日志信息
$log_type String 是 日志类型

实例：

$log = $this->getUtil('log');
$log->write('测试的信息', 'DEBUG');
