第2章 2.4 配置控制器文件

配置文件：
  • indexController.php文件（web/controller/文件夹下），默认配置后缀名称Controller,所以需要添加Controller作为你的Controller类名
  • $initphp_list是一个Action的白名单，默认执行是run()（不需要放进白名单中），其他都需要放入白名单
  • run() 函数是默认执行的函数，在这个函数中，打印出Hello World
  • 默认访问您的http://127.0.0.1/index.php?c=index&a=run
class indexController extends Controller {
	
	public $initphp_list = array('test'); 
	
	public function run() {
		echo 'Hello World';
	}
}
